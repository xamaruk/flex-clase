Selectores anidados y adyecentes en CSS

Ejemplo de selector adyacente:

Con este html:


        <div class="disparador">
            <p>Dispárame</p>
        </div>
        <div class="emergente">
            <p>Pam!</p>
        <div>
        
Y este css

        .disparador + .emergente {
            color: white;
        }

Afectamos al elemento con la clase "emergente" cuando está a continuación de "disparador":

Dispárame

Pam!

Pero si está después de otro elemento no le afecta ese selector:

Sigue el contenido del artículo

Estoy fuera del disparador

Ejemplo de selector anidado:

Con este html:


    <div class="disparador">
    <p>Dispárame</p>
        <div class="emergente">
        <p>Pam!</p>
        <div>
        </div>
        
Y este css

        .disparador > .emergente {
        color: lime;
        }

Afectamos al elemento con la clase "emergente" cuando está dentro, como hijo directo, de "disparador":

Dispárame

Pam!

Pero si no es hijo directo (hay otro u otros elementos contenedores ente ambos) no le afecta ese selector:

Soy emergente, pero no hijo directo de disparador

Cuando queremos un selector que afecte a cualquier elemento "emergente" dentro de un elemento "disparador", sin importar el nivel de anidación usamos .disparador .emergente {}
