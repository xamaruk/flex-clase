Ejemplos de Fondo con CSS

En esta cabecera aplicamos el color de fondo con transparencia:


    .cabecera {
        font-size: 2rem;
        text-align: center;
        background-color: #bd1616c7;
    }

Imagen de fondo

Las imágenes de fondo por defecto se repiten a lo largo de su contenedor, además al elemento con la clase principal le añadimos un color de fondo con transparencia a las secciones para que su texto sea más legible sobre la imagen de fondo:


    body {
        background-image: url('../imgs/fondo.jpg');
    }
    .bloque {
        margin: 3rem 0;
        padding: 2rem;
        border: 1px solid;
        background-color: #fff9;
    }

Para controlar como se repite la imagen de fondo tenemos la propiedad background-repeat, por defecto tiene el valor repeat, que hace que la imagen se repita a lo ancho y alto del contenedor para cubrir todo el elemento.
Otros valores posibles son:

  • no-repeat: para que no se repita y se muestre una sola vez
  • repeat-x: para que se repita solo en el eje horizontal
  • repeat-y: para que se repita en el eje vertical

Tamaño de imagen de fondo

La propiedad background-size permite modificar el tamaño de la imagen insertada. Podemos definir valores en píxeles para el ancho y alto, en porcentajes, o usar palabras clave como contain y cover.
A nuestra clase principal le añadimos ahora estas propiedasdes:


    .principal {
        background-image: url(../imgs/playa4.jpg);
        background-repeat: no-repeat;
        background-size: 800px;
    }

Posición de la imagen de fondo

La propiedad background-position permite definir donde ubicar la imagen en el contenedor. Puede ser con valores en píxeles (o cualquier unidad válida en CSS), porcentajes, o las palabras clave top, center, bottom, right, left

Una forma rápida de hacer que la imagen se quede centrada en el contenedor sobre los dos ejes es añadiendo:


    .principal {
        background-position: center;
    }

Efecto parallax

Este efecto se consigue haciendo que el contenido del elemento pase por encima de su imagen de fondo, que permanece fija mientras hacemos scroll sobreel elemento. La propiedad que usamos es background-attachment, y sus valores son fixed, para conseguir ese efecto, o scroll para que la imagen se desplace con el elemento (este es el valor por defecto).

Hay que tener en cuenta que esta propiedad entra en conflicto con la propiedad de background-position sobre el eje vertical y habrá que trabajar con una de las dos únicamente:


    .principal {
        background-attachment: fixed;
    }

Galería de imágenes en CSS

Podemos crear una galería de imágenes en CSS sin usar elementos img, usando solamente elementos div controlados por clases y atributos id de CSS para mostrar las imágenes.

El html inicial es:


    <article class="galeria">
        <div class="item-galeria" id="foto_1"></div>
        <div class="item-galeria" id="foto_2"></div>
        <div class="item-galeria" id="foto_3"></div>
        <div class="item-galeria" id="foto_4"></div>
    </article>

El CSS para dar formato a la galería incial es este:


    .item-galeria {
        width: 40%;
        height: 350px;
        border: 1px solid;
        background-color: #e3af4da3;
        margin: 2rem auto;
    }
    .galeria {
        display: flex;
        flex-wrap: wrap;
        justify-content: space-evenly;
    }

Para que cada contenedor muestre una imagen usamos su selector de id con CSS:


    #foto_1 {
        background-image: url('../imgs/playa1.jpg');
    }
    #foto_2 {
        background-image: url('../imgs/playa2.jpg');
    }
    #foto_3 {
        background-image: url('../imgs/playa3.jpg');
    }
    #foto_4 {
        background-image: url('../imgs/playa4.jpg');
    }

Y por último, ajustamos las propiedades de imagen de fon do que hemos visto antes para el tamaño y la posición sobre la clase de los elementos de galería, añadieno esto:


    .item-galeria {
        background-size: contain;
        background-repeat: no-repeat;
        background-position: center;
    }

Ajustes con imágenes de fondo

Otras propiedades y comportamientos disponibles con CSS para las imágenes de fondo.

Intercambio con efecto hover

Podemos generar un efecto de intercambio de imágenes al pasar el ratón por encima de una imagen usando CSS, aplicando el pseudo-selector hover. Esto evita usar JavaScript. Es un componente muy habitual a la hora de construir los menús de los sitios para crear los efectos de intercambio de imagen en cada item del menú.

Todo lo que tenemos que hacer es definir la imagen de fondo por CSS y añadir el pseudo-selector con la imagen que queremos intercambiar:


    .foto-patron {
        width: 40%;
        height: 400px;
        margin: 2rem auto;
        background-image: url('../imgs/patron1.jpg');
        background-position: center;
        background-repeat: no-repeat;
        background-size: contain;
        border: 1px solid;
        background-color: #e3af4da3;
    }
    .foto-patron:hover {
        background-image: url('../imgs/patron2.jpg');
    }

Ejemplo de menú con intercambio de imagen

Usamos este HTML y CSS iniciales:


    <nav class="menu">
        <ul class="items-menu">
            <li><a href="#" class="elem-menu">Elemento 1</a></li>
            <li><a href="#" class="elem-menu">Elemento 2</a></li>
            <li><a href="#" class="elem-menu">Elemento 3</a></li>
            <li><a href="#" class="elem-menu">Elemento 4</a></li>
            <li><a href="#" class="elem-menu">Elemento 5</a></li>
        </ul>
    </nav>
    
    .menu {
        background-color: #71d0cc9e;
        padding: 2rem 0;
    }
    .items-menu {
        display: flex;
        flex-wrap: wrap;
    }
    .items-menu li {
        width: 20%;
        list-style: none;
    }
    .elem-menu {
        text-decoration: none;
        text-align: center;
        display: block;
        background-color: antiquewhite;
        padding: 1rem;
        border: 1px solid;
        color: orange;
    }

A partir de aquí añadimos el CSS para mostrar la imagen en cada uno de los elementos del menú, y el pseudo-selector hover para indicar la imagen de intercambio, además de todo el formato adicional que queramos aplicar para ese selector.


    .elem-menu {
        background-image: url('../imgs/flecha-amarilla.png');
        background-size: 40px;
        background-repeat: no-repeat;
        background-position: 210px center;
    }

    .elem-menu:hover {
        background-image: url('../imgs/flecha-roja.png');
        color: red;
    }

Posición de la imagen de fondo respecto a su contenedor

El modelo de caja de HTML define 3 límites posibles para un contenedor: el que establece la posición externa de su borde, si lo tiene, el que establece el inicio de la zona de padding, que corresponde con el lado interior del borde, y el que establece la zona de contenido a partir del padding que pueda llevar ese elemento. Respectivamente son los valores border-box, padding-box content-box, y pueden ser definidos por la propiedad background-origin. Partimos de estos 3 elementos con el mismo formato y misma imagen de fondo:


    <div class="origen" id="origen_1"></div>
    <div class="origen" id="origen_2"></div>
    <div class="origen" id="origen_3"></div>
    .origen {
        width: 32%;
        height: 400px;
        background-color: #5b8ab39e;
        border: 50px solid #f5f5dc80;
        margin: 2rem auto;
        padding: 25px;
        background-image: url('../imgs/patron1.jpg');
        background-repeat: no-repeat;
        background-size: 270px;
    }

Para cada uno de ellos definimos un valor diferente para el origen de la imagen de fondo:


    #origen_1 {
        background-origin: border-box;
    }
    #origen_2 {
        background-origin: padding-box;
    }
    #origen_3 {
        background-origin: content-box;
    }

Degradados en fondos con CSS

Podemos aplicar valores de color no sólidos a los elementos de fondo de las etiquetas html con la siguiente propiedad:


    pre {
        background: linear-gradient(to bottom,  rgb(6,224,224) 0%,
        rgb(184,221,221) 50%,rgb(244,205,97) 100%);
    }

Para definir los valores de degradado de forma visual es útil tener alguna web de referencia donde generar el código CSS y copiarlo a nuestro documento, por ejemplo Colorzilla Gradient Generator
